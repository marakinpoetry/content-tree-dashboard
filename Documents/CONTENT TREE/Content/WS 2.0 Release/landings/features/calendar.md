UNIQUE VALUE PROPS:

7. Auto-rescheduling suggestions when plans change

9. iCalendar export to Google, Apple, Outlook, Mozilla

                                              CALENDAR FEATURES (from FAQ)
═══════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

DATA POPULATION:
• Automatic display of completion dates for all projects and tasks with defined deadlines
• Employee and client birthdays included
• Archived projects disappear from calendar view

ACCESS CONTROL:
• Personalized visibility — users only see deadlines for projects/tasks they can access
• Plus birthdays of colleagues from shared projects

COLOR CODING SYSTEM:
• Brown — Projects
• Blue/Gray — Tasks
• Green — Vacation days
• Pink variations — Birthdays and sick leave
• Red — Overdue open items (changes to gray when closed)
• Gray — Closed tasks

INTERACTIVE CAPABILITIES:
• Drag-and-drop editing: reschedule tasks by dragging events to new dates
• Save changes with optional notifications to assignees and managers
• Requires project leadership or task authorship permissions

HOVER FUNCTIONALITY:
• Hovering over task markers highlights all execution days
• Displays project name, task name, author, and assignee

FILTERING:
• Dynamic filters by project, author, and assignee
• Quick filters for vacations, sick leave, birthdays
• View only your own task deadlines

SYNC:
• iCalendar format supported
• Google Calendar, Microsoft Outlook, Apple Calendar, Mozilla Calendar
• Export link: webcal:// or http:// format

LIMITATIONS:
• Calendar displays only deadline-related information
• Personal date entries require external calendar integration
