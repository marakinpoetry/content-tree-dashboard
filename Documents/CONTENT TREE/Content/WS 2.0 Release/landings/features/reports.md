FOUR REPORT TYPES:

1. REPORT BY TASKS
   - Task completion rates and trends
   - Overdue items tracking
   - Status distribution analysis
   - Progress metrics
   - Filter by: project, assignee, date range, custom fields

2. REPORT BY TIME & COSTS (Витрати)
   - Hours logged per task/project/person
   - Costs incurred
   - Billable vs non-billable breakdown
   - Plan vs Actual comparison
   - Hourly billing rate calculations

3. REPORT BY PEOPLE
   - Individual performance metrics
   - Team workload distribution
   - Productivity analysis
   - Connects with Day Plan for daily insights
   - Who's overloaded, who has capacity

4. REPORT BY DAYS
   - Daily activity breakdown
   - Tasks completed per day
   - Time logged per day
   - Activity peaks and patterns
   - Weekly productivity flow

EXPORT FORMATS:
• CSV — for custom analysis, accounting software
• Excel — full data with formatting
• PDF — professional reports for clients/stakeholders

KEY FEATURES:
• Reports as project tabs — add any report to project navigation
• Saved report views — save filters, run same report repeatedly
• Aggregated client reports — roll up data using project hierarchy
• Real-time activity analytics — see who's working on what now
• Project statistics widget — KPIs on dashboard

BILLING & FINANCE:
• Billable / Non-billable time separation
• Hourly rates per person or role
• Automatic cost calculations
• Client invoicing support
• Budget tracking (plan vs actual)

DAY PLAN INTEGRATION:
• Personal daily planning feeds into reports
• "Report by People" shows how planned work translates to actual output
• "Report by Days" reflects Day Plan execution
• Workload forecasting data flows to team analytics
• Individual productivity visible through Day Plan → Reports connection

TARGET AUDIENCES (Tab Section):
• Agencies — client billing, project profitability, resource allocation
• Consulting — hourly billing, utilization rates, client reporting
• IT Teams — sprint metrics, team velocity, capacity planning
• Service Companies — time tracking, cost analysis, client invoicing

USE CASES:
• Client invoicing — accurate time reports with billable hours
• Project profitability — compare costs vs revenue
• Resource planning — who's available, who's overloaded
• Performance reviews — individual contribution metrics
• Budget tracking — plan vs actual throughout project
• Stakeholder reporting — PDF exports for presentations

TONE: Professional, data-driven, focused on actionable insights and accurate billing
