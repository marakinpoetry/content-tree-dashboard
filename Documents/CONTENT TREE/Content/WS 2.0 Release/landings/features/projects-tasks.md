KEY MESSAGES:
• Focus on clarity and team synchronization
• Highlight flexibility (create tasks from anywhere)
• Emphasize organization at scale (project hierarchy)
• Stress ease of use (inline editing, drag & drop)

UNIQUE VALUE PROPS:
1. Task creation from 8+ sources (button, keyboard, email, chat, comments, calendar, kanban, existing tasks)
2. Unlimited subtask hierarchy with visual indicators
3. Advanced dependencies (5 types: blocking, informational, start-after, start-start, finish-finish)
4. Project hierarchy for enterprise organization
5. Templates for standardization
6. Saved views for personalized workspace
7. Checklists with progress tracking
8. Full customization without complexity
9. Day Plan — personal daily planner with timeline, Drag&Drop, Google/Outlook sync, workload forecasting

DAY PLAN FEATURES (from comparison table):
• Personal planner with hourly timeline — all tasks for the day on one screen
• Drag & Drop editing — move tasks like cards
• Timezone support — for distributed teams
• Google & Outlook integration — personal and work events in one calendar
• Workload forecasting — warns when day is overbooked
• Auto-rescheduling — if one task delays, system helps rebuild the day
